## ☁️ Free Hosting Options

### Option 1 — GitHub Pages (Recommended)

1. Push this repo to GitHub
2. Go to **Settings → Pages**
3. Set source to `main` branch, root folder
4. Your site will be live at:  
   `https://your-username.github.io/leela-farm`

### Option 2 — Netlify Drop

1. Go to [netlify.com/drop](https://netlify.com/drop)
2. Drag and drop `leela-farm.html`
3. Get an instant live URL in seconds

### Option 3 — Vercel

```bash
npm install -g vercel
vercel --name leela-farm
```

## 🔍 SEO & Google Search

The website includes the following SEO optimizations so it appears when people search for **"Lila Farmhouse"** or **"Leela Farm Panvel"**:

- Meta `title`, `description`, and `keywords` tags
- OpenGraph tags for social sharing
- Keywords include: `Leela Farm`, `Lila Farmhouse`, `farmhouse Panvel`, `farmhouse Maharashtra`, `weekend getaway near Mumbai`

### To appear on Google:

1. Deploy the site (GitHub Pages or Netlify)
2. Visit [Google Search Console](https://search.google.com/search-console)
3. Add your site URL → **URL Inspection → Request Indexing**
4. Google will index your site within 3–7 days
